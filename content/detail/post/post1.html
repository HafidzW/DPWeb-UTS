Kami menjawab sebagai berikut

Apa itu Linux?
Linux yang sering disebut masyarakat adalah platform Linux sebagai perbandingan untuk platform Windows. Linux yang dimaksudkan adalah sistem operasi bernama GNU/Linux sebagai platform di mana software bisa berjalan di dalamnya. Linux juga berarti nama untuk kernel "Linux", kernel yang berlogo penguin tersenyum, komponen inti sistem operasi GNU/Linux yang bisa diunduh di http://kernel.org. Ubuntu itu Linux, Fedora, openSUSE, Mageia, dan Slackware, juga disebut Linux. Singkatnya, Linux adalah sebutan untuk sistem operasi free yang bukan Windows yang memiliki kernel Linux yang dikembangkan secara open source dan mampu menggantikan Windows di kalangan pengguna akhir.

Apa kelebihan Linux?
Di antaranya berlisensi free software, dikembangkan secara open source, aman, stabil, bisa diperoleh gratis, bisa disebarluaskan gratis, serta memiliki dukungan software yang sangat banyak. Dibandingkan Windows, secara internal (jeroan sistem), Linux lebih aman karena sejak awal sudah didesain multiuser-multitasking serta memiliki banyak prinsip keamanan ketat (superuser, advanced permission, system lock-down, dll.) yang tidak dimiliki oleh Windows. Linux dikenal "kebal virus" karena sisi internal (jeroan) ini sehingga lebih sulit bagi pembuat virus/malware ditambah lebih sulit lagi menyebar ke varian Linux lain karena Linux banyak variannya. Sisi "real" Linux dari pandangan pengguna akhir, biasanya orang pakai Linux karena ingin hidup tanpa virus tanpa antivirus. Mereka dapat itu dengan Linux. Dari sisi eksternal, pengguna Linux ahli dalam jumlah besar mampu mendeteksi kesalahan sistem dan mengirim patch (karena sifatnya open source) ke pengembang Linux sehingga masalah keamanan cenderung lebih cepat selesai sedangkan tidak ada yang seperti itu di Windows (semua dilakukan oleh Microsoft sendiri).

Adakah virus di Linux?
Ya, contohnya Staog. Ia adalah virus pertama yang ditemukan di Linux.

Apa saja aplikasi di Linux?
Aplikasi populer di Linux misalnya GIMP (menggantikan Photoshop), Inkscape (menggantikan CorelDRAW), Scribus (menggantikan InDesign), Blender (menggantikan 3D Studio Max), LibreOffice (menggantikan Microsoft Office), Rhythmbox (menggantikan Winamp), VLC (menggantikan Media Player Classic), dan seterusnya. Linux adalah platform, Windows adalah platform juga. Masingmasing punya pendukung sendiri-sendiri karena itu masing-masing memiliki program yang berbeda. Catatan: aplikasi Windows (di level binary .EXE) bisa berjalan di Linux dengan bantuan program Wine

Bagaimana cara mendapatkan software Linux?
Anda tidak menggunakan Linux, kecuali menggunakan distribusi Linux (distro). Anda menggunakan Linux dengan memilih salah satu distro Linux yang sudah ada semisal Ubuntu, Linux Mint, Fedora, openSUSE, atau Slackware. Cara mendapatkan software bergantung distro yang Anda pakai. Ubuntu misalnya, punya semacam Android Market atau Apple AppStore, untuk menginstal program dengan cari dan klik (wajib punya internet). Distro seperti openSUSE dan Fedora memiliki cara serupa (juga wajib internet). Informasi mutakhir semua distro ada di http://distrowatch.com

Bagaimana cara menggunakan Linux?
Seperti poin sebelumnya, dengan menggunakan salah satu distro Linux. Misalnya Ubuntu, maka pelajari bagaimana cara menggunakan Ubuntu. Instal ke dalam Ubuntu Anda program GIMP, Inkscape, Scribus, atau Blender yang disebut di atas. Dengan itu, Anda belajar menggunakan suatu OS Linux yang menggantikan Windows sebagaimana yang Anda maksudkan. Unduh ISO Ubuntu (bukan ISO Linux, tapi ISO Ubuntu) lalu instal di komputer dan gunakan sehari-hari. Jika mau Fedora, maka yang diunduh adalah ISO Fedora. Dan seterusnya untuk openSUSE dan yang lain-lain. Intinya, pilih salah satu distro dan pelajarilah. Pemula direkomendasikan menggunakan Ubuntu untuk pemakaian pertama.
